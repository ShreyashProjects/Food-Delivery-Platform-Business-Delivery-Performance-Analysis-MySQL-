# Food Delivery Platform – Business & Delivery Performance Analysis (MySQL)

## Project Overview

This project analyzes a food delivery platform dataset similar to services like Swiggy or Zomato.

The goal of the project is to extract meaningful business insights using SQL queries related to:

- Customer ordering behavior
- Restaurant performance
- Delivery efficiency
- Revenue analysis
- Peak ordering hours

This project demonstrates how SQL can be used for real-world business data analysis.

--------------------------------------------------

## Database Tables

1. customers
2. restaurants
3. delivery_agents
4. orders
5. order_items

## Question to be Answered 
# Food Delivery Data Analysis (MySQL Project)

This project analyzes a food delivery platform dataset to understand customer behavior, restaurant performance, delivery efficiency, and order trends using SQL.

---

## Basic Analysis

1. What is the total number of orders placed?

2. What is the total revenue generated from all delivered orders?

3. What is the average order value?

---

## Customer Behavior Analysis

4. Who are the top 5 customers based on total spending?

5. Which customers have placed more than 3 orders?

6. What is the ratio of new customers to returning customers?

---

## Restaurant Insights

7. Which restaurant has generated the highest revenue?

8. What is the average rating of restaurants in each city?

9. Which cuisine type has the highest number of orders?

---

## Delivery Performance

10. What is the average delivery time for each city?

11. Which delivery agents have the fastest average delivery times?

12. Which orders were delivered late or marked as delayed?

---

## Time-Based Analysis

13. During which hour of the day are the most orders placed (peak order hour)?

14. How many orders are placed each day?

15. What is the monthly revenue trend over time?
